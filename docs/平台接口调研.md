## 一、用户需求

用户当前使用的 API 服务：

| 平台 | 产品类型 | 用途 |
|------|---------|------|
| **MiniMax** | Token Plan | 通用大模型调用 |
| **MiniMax** | Coding Plan | AI 编程工具 |
| **小米 MiMo** | Token Plan | 通用大模型调用 |
| **京东云** | Coding Plan | AI 编程工具 |

**目标**：二开一个 Windows 托盘工具，实时监控各平台的 API 余额和用量。

## 三、各平台接口详情

### 3.1 MiniMax Token Plan

**接口地址**：
```
GET https://www.minimaxi.com/v1/token_plan/remains
```

**请求头**：
```
Authorization: Bearer YOUR_API_KEY
```

**响应示例**：
```json
{
  "model_remains": [
    {
      "model_name": "general",
      "current_interval_remaining_percent": 99,
      "current_weekly_remaining_percent": 100,
      "current_interval_usage_count": 0,
      "current_weekly_usage_count": 0
    },
    {
      "model_name": "video",
      "current_interval_remaining_percent": 66,
      "current_weekly_remaining_percent": 95
    }
  ],
  "base_resp": {
    "status_code": 0,
    "status_msg": "success"
  }
}
```

**响应字段说明**：

| 字段 | 类型 | 含义 |
|------|------|------|
| `model_remains` | array | 所有模型的配额数据 |
| `model_name` | string | 模型名称（general/video 等） |
| `current_interval_remaining_percent` | number | 5小时窗口剩余百分比 |
| `current_weekly_remaining_percent` | number | 周剩余百分比 |
| `current_interval_usage_count` | number | 5小时窗口已用次数 |
| `current_weekly_usage_count` | number | 周已用次数 |

**curl 测试**：
```bash
curl -s "https://www.minimaxi.com/v1/token_plan/remains" \
  -H "Authorization: Bearer YOUR_KEY"
```

### 3.2 MiniMax Coding Plan

**接口地址**：
```
GET https://www.minimaxi.com/v1/api/openplatform/coding_plan/remains
```

**请求头**：
```
Authorization: Bearer YOUR_API_KEY
```

**响应示例**：
```json
{
  "model_remains": [
    {
      "model_name": "deepseek-chat",
      "current_interval_remaining_percent": 85,
      "current_interval_usage_count": 150,
      "current_interval_total_count": 1000
    }
  ],
  "current_interval_total_count": 1500,
  "current_interval_usage_count": 100,
  "base_resp": {
    "status_code": 0,
    "status_msg": "success"
  }
}
```

**响应字段说明**：

| 字段 | 类型 | 含义 |
|------|------|------|
| `current_interval_total_count` | number | 总额度 |
| `current_interval_usage_count` | number | 已使用额度 |
| `model_remains` | array | 各模型详细配额 |

### 3.3 小米 MiMo Token Plan（API Key 方式）

**接口地址**：
```
GET https://token-plan-cn.xiaomimimo.com/v1/usage
```

**请求头**：
```
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

**预期响应**：
```json
{
  "used": 1000000000,
  "total": 4100000000,
  "remaining": 3100000000,
  "expires_at": "2026-07-01T00:00:00Z"
}
```

**curl 测试**：
```bash
curl -X GET "https://token-plan-cn.xiaomimimo.com/v1/usage" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

### 3.4 小米 MiMo Token Plan（Cookie 方式）

**接口地址**：
```
GET https://platform.xiaomimimo.com/api/v1/tokenPlan/usage
```

**请求头**：
```
Cookie: api-platform_serviceToken="YOUR_COOKIE"
Accept: application/json
User-Agent: cc-switch/1.0
```

**响应示例**：
```json
{
  "data": {
    "usage": {
      "items": [
        {
          "name": "plan_total_token",
          "used": 1000000000,
          "limit": 4100000000,
          "percent": 0.24
        },
        {
          "name": "compensation_total_token",
          "used": 0,
          "limit": 0,
          "percent": 0
        }
      ],
      "percent": 0.24
    }
  }
}
```

**响应字段说明**：

| 字段 | 类型 | 含义 |
|------|------|------|
| `name` | string | 积分类型（plan_total_token/compensation_total_token） |
| `used` | number | 已使用 Credits |
| `limit` | number | 总 Credits |
| `percent` | number | 使用百分比 |

**⚠️ 注意**：Cookie 有效期约 1 天，需要定期刷新。

3. **联系平台**：申请内部 API 接口

**控制台入口**：https://docs.jdcloud.com/cn/jdaip/PackageOverview

## 四、二开架构设计

### 4.1 整体架构

```
┌─────────────────────────────────────────────────────────────────┐
│                    Windows 托盘程序 (Python)                      │
├─────────────────────────────────────────────────────────────────┤
│  主进程                                                          │
│  ├── tray.py          → 托盘图标 + 右键菜单                        │
│  ├── config.py        → 多平台配置管理                            │
│  └── settings.py      → Web 设置界面（本地 HTTP 服务）              │
├─────────────────────────────────────────────────────────────────┤
│  平台适配层                                                      │
│  ├── providers/                                                 │
│  │   ├── base.py      → 抽象基类                                 │
│  │   ├── minimax_token.py    → MiniMax Token Plan               │
│  │   ├── minimax_coding.py   → MiniMax Coding Plan              │
│  │   ├── mimo_api.py         → 小米 MiMo (API Key)               │
│  │   ├── mimo_cookie.py      → 小米 MiMo (Cookie)                │
│  │   └── jdcloud.py          → 京东云（待定）                      │
│  └── api.py           → HTTP 请求封装                            │
├─────────────────────────────────────────────────────────────────┤
│  数据层                                                          │
│  ├── cache.py         → 本地缓存（避免频繁请求）                    │
│  └── history.py       → 历史记录（可选，用于趋势图）                │
└─────────────────────────────────────────────────────────────────┘
```
